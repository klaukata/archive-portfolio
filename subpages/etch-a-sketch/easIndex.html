purpose and goal

The goal was to create a basic "sketchbook" that operates like a mouse-controlled Etch-a-Sketch: hovering over and pressing a square would color it with a random or fixed color. Changing the size of given squares would reset any previous modifications.

This project is a slightly altered version of
The Odin Project's assignment, where the final result would be a bit less complex since it required just a hover effect present.

general requirements

As always, apart from the bare minimum that the generated website must be responsive. I had a few requirements in mind to be able to call this project a success:

  • The resulting website must match the design on a Figma file, that I created for it
  • Add an animation that makes an image of what user/computer chose bigger

challenges

Surprisingly, for me, I had some issues with scopes of variables. Variables that were defined outside of the function were not 'found' by my code editor, so at my first attempt, I nested all of my code inside of the one function. After I decided to rewrite my code this problem never surfaced again, so I am not completely sure what I did wrong the first time. However, I suspect that I must have simply misspelled variable names.
