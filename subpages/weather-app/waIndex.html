purpose and goal

My goal was to learn how asynchronous code works by learning how to implement data fetching. Is there a better way to learn how to do something, other than practicing it? Honestly, I don't think so.

user stories:

  • is responsive
  • matches previously designed UI
  • has a button that switches all temperatures from Celcius to Fahrenheit and vice versa
  • includes a heading that displays "Today" or "Tomorrow" depending on the circle that was clicked
  • displays an 8-hour forecast
  • 8-hour forecast' circles are clickable. Once clicked, the div changes its colors, weather details section, and elements that display the current temperature and weather icon
  • "weather details" section contains eight divs, each displaying: sunrise, sunset, precipitation, humidity, UV index, visibility and "feels like" temperature
  • background-color changes depending on a temperature
